# Publisher Backend

O **Publisher Backend** (`packages/publisher-backend`) é uma plataforma de criação de conteúdo para LinkedIn. Gerencia rascunhos de posts, geração de conteúdo com IA, renderização de PDFs de carrossel, upload de imagens e publicação agendada.

---

## O Que Faz

- **Gestão de Posts**: CRUD completo para posts LinkedIn com anexos de mídia (imagens, PDFs)
- **Geração de Conteúdo IA**: Gerar posts e estruturas de carrossel a partir de prompts usando Gemini
- **Renderização de Carrossel**: Criar PDFs de carrossel otimizados para LinkedIn (1080x1350px) com 5 temas integrados
- **Upload de Imagem**: Fluxo de upload multi-etapa via API Voyager Ambry do LinkedIn
- **Publicação Agendada**: Agendador em background auto-publica posts em horários especificados

| `warm-creative` | Tons quentes com layout criativo |

---

## Publicação Agendada

O backend executa um worker em background (intervalo de 30 segundos) que verifica posts com `status: 'scheduled'` e `scheduledAt <= now`, então os publica no LinkedIn via Gateway.

---

## Variáveis de Ambiente

```bash
# Integração LLM
NINE_ROUTER_API_KEY=sua_chave_api
NINE_ROUTER_BASE_URL=http://localhost:20128/v1
NINE_ROUTER_MODEL=google/gemini-3.5-flash

# Conexão Gateway
LINKEDIN_SERVICE_URL=http://localhost:4000/graphql

# Porta do Servidor
PORT=3001
```

## Stack Tecnológico

| Componente | Tecnologia |
| :--- | :--- |
| Framework | Express (Node.js, TypeScript) |
| Banco de Dados | Prisma + SQLite (`better-sqlite3`) |
| LLM | Google Gemini via API compatível OpenAI |
| PDF | Puppeteer (Chrome headless) |
| Upload de Imagem | API LinkedIn Voyager Ambry |
| Agendador | Worker `setInterval` em server.ts |
