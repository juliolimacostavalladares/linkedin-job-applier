# API Reference

Welcome to the LinkedIn Gateway REST API reference. This API allows you to search jobs, pull dynamic easy-apply questionnaires, submit applications, manage feed posts, and retrieve user profiles directly.

---

## Jobs API

<Cards>
  <Card title="List Jobs" href="/docs/api/jobs/listJobs">
    `GET /api/jobs` - Retrieve a list of Easy-Apply jobs matching search filters.
  </Card>
  <Card title="Get Job Details" href="/docs/api/jobs/getJobDetail">
    `GET /api/jobs/{id}` - Fetch detailed job description and requirements.
  </Card>
  <Card title="Get Apply Form" href="/docs/api/jobs/getApplyForm">
    `GET /api/jobs/{id}/apply-form` - Extract dynamic easy-apply questionnaire URN fields.
  </Card>
  <Card title="Submit Application" href="/docs/api/jobs/submitApplication">
    `POST /api/jobs/{id}/apply` - Submit form responses and resume uploads.
  </Card>
</Cards>

---

## Posts API

<Cards>
  <Card title="Create Post" href="/docs/api/posts/createPost">
    `POST /api/posts` - Publish articles, text, images, or documents to the feed.
  </Card>
  <Card title="Delete Post" href="/docs/api/posts/deletePost">
    `DELETE /api/posts/{id}` - Delete a published post using its share ID.
  </Card>
</Cards>

---

## Profile API

<Cards>
  <Card title="Get Profile Info" href="/docs/api/profile/getProfileInfo">
    `GET /api/profile/info` - Download, parse, and structure user profile into JSON.
  </Card>
  <Card title="Get Resume PDF" href="/docs/api/profile/getResumePdf">
    `GET /api/profile/resume-pdf` - Retrieve the member's profile summary in PDF binary format.
  </Card>
</Cards>
