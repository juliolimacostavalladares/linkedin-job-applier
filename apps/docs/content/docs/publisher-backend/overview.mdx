# Publisher Backend

The **Publisher Backend** (`apps/linkedin-publisher-backend`) is a content creation platform for LinkedIn. It handles post drafting, AI content generation, carousel PDF rendering, image uploads, and scheduled publishing.

---

## What It Does

- **Post Management**: Full CRUD for LinkedIn posts with media attachments (images, PDFs)
- **AI Content Generation**: Generate posts and carousel structures from prompts using Gemini
- **Carousel Rendering**: Create LinkedIn-optimized carousel PDFs (1080x1350px) with 5 built-in themes
- **Image Upload**: Multi-step upload flow via LinkedIn's Voyager Ambry API
- **Scheduled Publishing**: Background scheduler auto-publishes posts at specified times

## Scheduled Publishing

The backend runs a background worker (30-second interval) that checks for posts with `status: 'scheduled'` and `scheduledAt <= now`, then publishes them to LinkedIn via the Gateway.

## Environment Variables

```bash
# LLM Integration
NINE_ROUTER_API_KEY=your_api_key
NINE_ROUTER_BASE_URL=http://localhost:20128/v1
NINE_ROUTER_MODEL=google/gemini-3.5-flash

# Gateway Connection
LINKEDIN_SERVICE_URL=http://localhost:4000/graphql

# Server Port
PORT=3001
```
