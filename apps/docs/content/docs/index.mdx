# LinkedIn Gateway API

Welcome to the official documentation for the **LinkedIn Gateway Service**. This microservice is a secure, decoupled gateway that exposes both **REST** and **GraphQL** endpoints to interface dynamically with LinkedIn Voyager APIs. It allows developers to search jobs, submit applications, manage posts, and parse profile data using browser session cookies.

## Architectural Highlights

- **Dual-Interface support**: Query endpoints via high-efficiency GraphQL or standard OpenAPI/REST interfaces.
- **Dynamic Session Handling**: Does not store or fix credentials. Cookies and CSRF tokens are parsed from request headers.
- **Credential Redaction**: Middleware automatically sanitizes and redacts sensitive cookie strings from logs to prevent leaks.
- **Strict Typing**: Implemented entirely in TypeScript with strict compile-time type validation.
